API Documentation

In Zero, we offer our services, so every user can have access to our database. Therefore, we have created this API. It is quite simple to use, the user just needs to make a GET HTTP request with the data they need. Every object will be shown with JSON format.
Examples

If you want to get all the platforms

GET

If you want to get all the games with a rating between 3 and 4

GET
Then, all the documentation on the data that can be collected and the parameters by which it can be filtered will be shown.

Get a list of games

GET

QUERY PARAMETERS

ID Integer
name String
minRating Double between 0 and 5
maxRating Double between 0 and 5
year Integer

RESPONSE SCHEMA

ID Integer {Not empty}
slug String {Not blank}
name String {Not blank}
description String
released Date
rating Double between 0 and 5
image_background String {URL}

Get a list of progress

GET

QUERY PARAMETERS

ID Integer
gameName String
gameId Integer
minRating Double between 0 and 5
maxRating Double between 0 and 5
review Boolean
reviewId Integer
username String

RESPONSE SCHEMA

ID Integer {Not empty}
status String {Not blank}
rating Double {Not empty} (between 0 and 5)
finish_date Date
review Object (Review)
game Object (Game)
username String {Not blank}
